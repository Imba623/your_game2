# Твоя игра

Браузерная интеллектуальная игра в стиле телевикторины для трех игроков за одним компьютером.

## Возможности

- Три игрока с индивидуальными клавишами для ответа (A, L, ПРОБЕЛ)
- Два основных раунда с темами и вопросами разной стоимости
- Специальные типы вопросов: "Кот в мешке" и "Аукцион"
- Финальный раунд с секретными ставками
- Сохранение состояния игры при обновлении страницы
- Режим разработчика для упрощения тестирования
- Адаптивный дизайн

## Установка и запуск

```bash
# Установка зависимостей
npm ci

# Запуск в режиме разработки
npm run dev

# Сборка статики
npm run build

# Запуск тестов
npm test
